# Assets Directory

This directory contains app icons and splash screens for the mobile app.

## Required Files

For production deployment, you'll need to add:

- `icon.png` (1024x1024) - App icon for all platforms
- `splash.png` (1242x2436) - Splash screen image
- `adaptive-icon.png` (1024x1024) - Android adaptive icon
- `favicon.png` (16x16 or 32x32) - Web favicon

## Temporary Setup

For now, the app will use Expo's default icons. To customize:

1. Add your icon files to this directory
2. Update the paths in `app.json` if needed
3. Rebuild the app

## Icon Guidelines

- Use high-quality PNG files
- Follow platform-specific design guidelines
- Ensure good contrast and readability
- Test on different background colors
